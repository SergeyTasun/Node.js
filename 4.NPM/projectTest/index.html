Создайте каталог для проекта с произвольным названием. Сгенерируйте файл package.json. Установите в проект пакет live-server (применяется при разработке) с добавлением в файл package.json. Добавьте в корень проекта файл index.html с произвольным содержимым. Запустите live-server. Файл для запуска находится в подкаталоге проекта: ./node_modules/.bin/ В результате должен открыться браузер и отобразить файл index.html.
